# 📝 Assignment Task Manager App

A robust and beautifully designed **Task Management Application** built using **Flutter**. This application helps users track, organize, and manage their daily tasks efficiently with real-time status updates, category-based filtering, and a clean user interface.

## 🚀 Features

- **User Authentication:** Secure Signup, Login, and State Persistence (SharedPreferences).
- **Password Recovery:** Seamless Forget Password flow via Email and OTP verification.
- **Task Management:** Create, Read, Update, and Delete (CRUD) tasks.
- **Status Dashboard:** Quick overview of tasks categorized by *New*, *In Progress*, *Completed*, and *Cancelled*.
- **Category Filtering:** View tasks fluidly using status-based ChoiceChips in the Categories tab.
- **Calendar & Timeline:** Organized calendar view to keep track of task creation dates.
- **Profile Management:** View and update user profile details seamlessly.

## 🛠️ Tech Stack & Packages

- **Frontend:** Flutter & Dart
- **State Management & Architecture:** Layered Architecture (Controller, Service, Model, View)
- **HTTP Requests:** Rest API integration (`http` package)
- **Local Storage:** `shared_preferences` for token and user data caching
- **UI Design:** Custom SVG Backgrounds (`flutter_svg`), Google Fonts, and Custom Widgets
